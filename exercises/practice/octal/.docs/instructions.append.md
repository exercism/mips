# Instructions append

## Registers

| Register | Usage     | Type    | Description                           |
| -------- | --------- | ------- | ------------------------------------- |
| `$a0`    | input     | address | null-terminated string of octal chars |
| `$v0`    | output    | integer | decimal value of input string         |
| `$t0-9`  | temporary | any     | used for temporary storage            |
