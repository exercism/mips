# Instructions append

## Registers

| Register | Usage        | Type    | Description                   |
| -------- | ------------ | ------- | ----------------------------- |
| `$a0`    | input        | integer | given number                  |
| `$a1`    | input/output | address | null-terminated result string |
| `$t0-9`  | temporary    | any     | for temporary storage         |
