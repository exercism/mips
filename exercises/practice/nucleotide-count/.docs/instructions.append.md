# Instructions append

## Registers

| Register | Usage        | Type    | Description                             |
| -------- | ------------ | ------- | --------------------------------------- |
| `$a0`    | input        | address | null-terminated input string            |
| `$a1`    | input/output | address | word array with final nucleotide-counts |
| `$t0-9`  | temporary    | any     | for temporary storage                   |
