# Instructions append

## Registers

| Register | Usage     | Type    | Description                                                                           |
| -------- | --------- | ------- | ------------------------------------------------------------------------------------- |
| `$a0`    | input     | integer | number                                                                                |
| `$v0`    | output    | integer | classification (`1` = `perfect`, `2` = `abundant`, `3` = `deficient`, `-1` = `error`) |
| `$t0-9`  | temporary | any     | used for temporary storage                                                            |
