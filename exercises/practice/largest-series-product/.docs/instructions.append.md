## Registers

| Register | Usage        | Type    | Description                   |
| -------- | ------------ | ------- | ----------------------------- |
| `$a0`    | input        | address | null-terminated digits string |
| `$a1`    | input        | integer | span                          |
| `$v0`    | output       | integer | largest series product        |
| `$t0-9`  | temporary    | any     | for temporary storage         |
